Obtener costos de envío de manera dinámica utilizando la API de Fast Mail http://www.fastmail.com.ar/.


== Description ==

Obtenga tarifas de envío con Fast Mail dinámicamente a través de la API Fast Mail para sus pedidos.

Con este complemento podrá obtener automáticamente las tarifas de envío de Fast Mail en la página del carrito y checkout con WooCommerce. Además le permite seleccionar a su cliente una sucursal de retiro. 

--------*******--------

 
 
== Installation ==

El plugin se mantiene actualizado para ser compatible con la última versión de Woocommerce y su instalación es bastante simple, consta de los siguientes pasos:

1) Instala y activa el plugin WooCommerce Fast Mail Shipping en tu tienda.

2) Ingresá a la opción “Ajustes” del menú Woocommerce, en el panel izquierdo de tu WordPress.

3) Acede a la pestaña “Envio”, si no hay zonas de envio, generarlas.

4) En el sub menú “Zonas de envio” selecciona Fast Mail”

5) Hace clic en “Activar este método de envio”

6) Completa los datos necesarios.

7) No olvides asignarle las caracteristicas para el envío a tus productos: Peso y dimensiones.

 
Una vez configurado con tus datos, el plugin ya está listo para calcular los costos de envío en el carrito de tu tienda.
